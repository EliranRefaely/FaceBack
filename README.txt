כמה מילים על הפרוייקט -
חיקוי פחות מוצלח לאתר של פייסבוק, קוראים לאתר FaceBack.
רק אחרי הרשמה לאתר אפשר לעלות פוסטים או לראות פוסטים (יש 2 - 3 פוסטים שאפשר לראות ללא הרשמה).

יש login and logout, מאחורה יש DB בשביל לשמור את ההרשמות ולוודא logins.

כמובן הכל נכתב ב-Flask, CSS and HTML.
נעזרתי ב-Bootstrap בשביל ה-CSS.
